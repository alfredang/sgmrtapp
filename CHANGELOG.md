# Changelog

All notable changes to SGMRT are documented here. The CI release pipeline reads the section
matching the version being submitted (falling back to `[Unreleased]`) and sets it as the
App Store "What's New" text. "What's New" only applies to updates of an already-released app.

## [Unreleased]

- Lines tab: browse every MRT line and see all its stations, with interchanges marked.

## [1.0]

- Shortest-route planning between any two MRT stations with estimated time, station count, and transfers.
- Route breakdown by line with per-segment timing.
- Built-in Singapore MRT system map.
- Live train service alerts and crowd density via LTA DataMall (when a key is configured).
